Put your audio files in these folders:

- audio/filler_talker/
- audio/native_1/
- audio/native_2/
- audio/non_native_1/
- audio/non_native_2/

Main-item recordings use the 4 main speaker folders.
Filler recordings use only the filler talker folder.

Example:
- audio/native_1/ha77v84840_1T_audio.webm
- audio/native_2/ftqpxk732r_1T_audio.webm
- audio/non_native_1/6bqg3s4g4q_1T_audio.webm
- audio/non_native_2/2j7bwre4ol_1T_audio.webm
- audio/filler_talker/7brghu4wnv_F1T_audio.webm
